# Trabalho de Estrutura de Dados – Pilha em Java

## Sobre

Esse trabalho foi feito para implementar a estrutura de dados pilha em Java.

Tem duas versões:

* Pilha Estática (com vetor)
* Pilha Dinâmica (com nós)

---

## Arquivos

* Main.java
* PilhaEstatica.java
* PilhaDinamica.java
* No.java
* PilhaOperacoes.java

---

## Funções básicas

* push (empilhar)
* pop (desempilhar)
* peek (ver topo)
* exibir elementos

---

## Observação

Trabalho feito para praticar pilha (LIFO) e entender melhor estruturas de dados em Java.
