# Implementação de Pilha em Java

Este projeto apresenta a implementação da estrutura de dados Pilha utilizando duas abordagens: estática e dinâmica.

##  Pilha Estática
- Utiliza vetor (array)
- Possui tamanho fixo
- Verifica se a pilha está cheia ou vazia

##  Pilha Dinâmica
- Utiliza nós encadeados
- Tamanho variável
- Cresce conforme a necessidade

##  Métodos implementados
- push: insere um elemento no topo
- pop: remove o elemento do topo
- peek: retorna o elemento do topo sem remover
- exibir: mostra todos os elementos da pilha

##  Execução
A classe Main.java contém exemplos de uso das duas implementações.

##  Conceito
A pilha segue o princípio LIFO (Last In, First Out), onde o último elemento inserido é o primeiro a ser removido.
